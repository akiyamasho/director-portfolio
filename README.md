# Portfolio Website

Portfolio website powered by [Gatsby](https://www.gatsbyjs.com)

### Setup

1. Install `gatsby` globally

```bash
pnpm install --global gatsby
```

2. Install dependencies

```
pnpm install
```

### Local development

1. Run `pnpm start`
2. Access http://localhost:8000 on your browser

### Deployment

`pnpm run deploy`
